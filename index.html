About Me

Hi, my name is Chukwu. I'm a full-stack JavaScript developer based in Lagos, Nigeria, with over 4 years of experience building web applications using HTML, CSS, JavaScript, and modern tools across both frontend and backend.

I studied Industrial Chemistry at the University of Ibadan, but my passion for technology and innovation led me into software development. I'm a fast learner who enjoys solving problems, building things from scratch, and working with other developers to ship great products.

I work mainly with tools like ReactJS, TailwindCSS, Node.js, and NestJS, and I've also worked with AWS, Docker, and PostgreSQL for deploying and managing full-stack applications. I focus on creating clean, responsive, user-friendly interfaces, and writing efficient backend code to support them.

In the past two years, I've collaborated on projects ranging from frontend design systems to API development and cloud deployments. I enjoy learning new technologies and constantly improving my skills to stay up-to-date in the fast-moving web ecosystem.
